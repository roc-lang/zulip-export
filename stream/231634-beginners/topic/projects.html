Richard Feldman (Aug 11 2021 at 01:54):

some other projects in the works besides the compiler internals @Folkert de Vries is working on and the editor improvements @Anton is working on:

  • revamping our canonical IR into a more efficient representation where we can allocate and deallocate parts of it on the fly, for use in the editor - @Lucas Rosa has been leading this
  • a built-in decimal type (128-bit fixed precision, and as fast as we could get it!) - @Jared Ramirez and @Brendan Hansknecht have been making this happen
  • a development backend that goes straight to machine code instead of using LLVM - @Brendan Hansknecht has done a ton of work on this, with others having pitched in here and there
  • automatically generated documentation from doc comments - currently on roc-lang.org but intentionally at a URL that's not referenced from the homepage :big_smile: - @Chad Stearns has been making great strides on this
  • eliminating the need for linking libc++ and libunwind, which is a prerequisite for a bigger project to dramatically improve linking speed and flexibility - @Joshua Hoeflich is gearing up to tackle this one next!

Richard Feldman (Aug 11 2021 at 01:54):

so yeah, lots of cool stuff going on!
