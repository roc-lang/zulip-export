Kilian Vounckx (Dec 09 2023 at 12:39):

I think your understanding of backpassing is wrong.

foo \k ->
   k + 1

Is equivalent to

k <- foo
k + 1

So if the last argument of a (higher order) function is itself a function, it can be used with backpassing style
