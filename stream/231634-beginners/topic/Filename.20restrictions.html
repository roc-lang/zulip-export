Richard Feldman (Dec 01 2022 at 06:22):

ah! Modules you can import need to start with a capital letter. This is so that the compiler can tell the difference between Foo.bar (look for bar to be exposed from a module named Foo) and foo.bar (look for a record field bar on a variable named foo)
