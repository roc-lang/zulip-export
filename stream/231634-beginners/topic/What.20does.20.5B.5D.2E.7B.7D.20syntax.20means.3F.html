Anton (Apr 04 2026 at 07:56):

Hi @gavr,
That's an excellent question, this is different from the old Roc.

  • := stands for a nominal type defintion. Nominal means two types only match if they refer to the same name (and module), Stdout in this case. I think old Roc always used structural types, where two types match if they have the same underlying structure.
  • [] is the empty tag union
  • .{ line! : Str => {} } is the associated items block, it contains associated values/methods/types for that nominal type

Feel free to ask more questions :)
