Luke Boswell (Dec 14 2023 at 10:59):

On my mobile so forgive me if the formatting is terrible.

# AddOne.roc
interface AddOne
    exposes [addOne]
    imports []

addOne = \x -> x+1

# main.roc
app ""
    package ... and other header stuff
    imports [ AddOne ]

expect AddOne.addOne 2 == 3
