Brendan Hansknecht (Dec 02 2023 at 17:26):

The error means that roc is expecting some sort of value, but instead things are just stored in variables. As a small example:

invalid:

main : U32
main =
    x = 7
    y = 3
    out = x + y

valid:

main : U32
main =
    x = 7
    y = 3
    x + y
