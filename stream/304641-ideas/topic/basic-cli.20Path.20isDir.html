I propose the addition of the following functions to basic-cli Path:

  • isDir : Path -> Bool
  • isFile: Path -> Bool
  • isSymLink: Path -> Bool
  • type: Path -> [IsFile, IsDir, IsSymLink] I used the Is prefix to prevent confusion/collision with the modules File and Dir
