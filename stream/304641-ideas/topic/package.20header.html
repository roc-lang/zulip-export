In response to this topic.

Instead of

package [ParserCore, ParserCsv, ParserStr]
    packages [
        JsonDecode from "https://…/json/…",
        CodePt, Segment from "https://…/unicode/…",
    ]

I would prefer:

package
    exposes [
        ParserCore, ParserCsv, ParserStr
    ]
    imports [
        JsonDecode from "https://…/json/…",
        CodePt, Segment from "https://…/unicode/…",
    ]

Because it is more self-evident
