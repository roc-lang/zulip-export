Jasper Woudenberg (Nov 07 2024 at 07:40):

For the 'location-string-as-a-function' approach Agus mentioned, an API like this might be nice to use:

# Translations.roc
module { locale } -> [hello, bye]

hello : { name : Str } -> Str
hello = \{ name } ->
    when locale is
        En -> "Hello, $(name)"
        Fr -> "Bonjour, $(name)"

bye : Str
bye =
    when locale is
        En -> "Bye!"
        Fr -> "Au revoir!"

Jasper Woudenberg (Nov 07 2024 at 07:49):

Jasper Woudenberg (Nov 07 2024 at 07:52):

A combined approach:

# Translations.roc
module { locale } -> [hello, bye]

hello : { name : Str } -> Str
hello = locale.hello

bye : Str
bye = locale.bye

Here the Locale type would (secretly) contain the translations for that locale, and the Translations.roc module would just be a wrapper for nice access to translations.
