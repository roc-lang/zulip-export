## Encode a range of a [List] of [U8] UTF-8 [code units](https://unicode.org/glossary/#code_unit)
## into a [Str]. The range is specified by a `start` index and a `count` of elements to encode.
##
## Returns `Err OutOfBounds` if the specified range exceeds the list's length.
## Returns `Err BadUtf8` if the given bytes within the specified range are invalid UTF-8.
## Returns `Ok ""` when given `[]` or a `count` of 0.
##
## Example:
## ```
## expect Str.toUtf8 "Hi there!" |> Str.fromUtf8Range { start : 0, count : 2 } == Ok "Hi"
## expect Str.toUtf8 "Hello, Roc!" |> Str.fromUtf8Range { start : 0, count : 0 } == Ok ""
## expect Str.toUtf8 "Hello, Roc!" |> Str.fromUtf8Range { start : 7, count : 3 } == Ok "Roc"
## expect Str.toUtf8 "Hello, Roc!" |> Str.fromUtf8Range { start : 12, count : 2 } |> Result.isErr
## ```
