according to Claude:

Yes, Raspberry Pi (newer models) are ARM64 (also called AArch64 or ARMv8).

Specifically:

  • Raspberry Pi 3, 4, 5, Zero 2 W - have 64-bit ARM processors (ARM64/AArch64)
  • Raspberry Pi 1, 2, Zero, Zero W - have 32-bit ARM processors (ARMv6 or ARMv7)

However, there's a catch: Raspberry Pi OS (the official operating system) ships in 32-bit mode by default, even on 64-bit hardware, for compatibility reasons. If you want to actually use ARM64, you need to:

  • Install the 64-bit version of Raspberry Pi OS, or
  • Use another 64-bit ARM distro like Ubuntu ARM64

So the hardware is ARM64 capable (on Pi 3+), but you may need to explicitly choose a 64-bit OS to take advantage of it.
