Package modules

  • share code organised in interface modules with multiple applications and platforms
  • basic file structure package "name" exposes [] packages {}
  • only those interface modules listed in exposes are included
  • generate docs with roc docs /package/*.roc currently broken zulip discussion
  • build URL package with roc build --bundle .tar.br /package/main.roc
  • import in Roc application with packages { .., parser: "<package URL>" }
  • working example using the parser package link
