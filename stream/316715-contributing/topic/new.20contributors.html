Luke Boswell (Oct 10 2025 at 03:37):

Welcome @PineyMarten @Heather Konecny, it's always great to meet new contributors.

We're in an interesting period at the moment with the Zig compiler foundations being laid -- it's definitely an exciting time to get involved with Roc.

Our GitHub Good First Issues label has fallen a little behind and isn't a great place to look right now. If you feel like reading through those and asking questions or making recommendations to help us update that could be helpful.

Generally, Zulip is the best place for general discussion -- so feel free to make a thread here and ask for clarification if you don't see an existing discussion or can't find any using the search functionality in Zulip.

The main focus right now if getting the new compiler ready for Advent of Code so we can try out the new syntax. Lot's of core work on the critical path that mostly Richard and Jared have been working on lately.

If you haven't already, checkout the Roc Playground -- I think that's a great way to learn more about the compiler stages and the intermediate representations in a human readable and interactive format. That's primarily only useful for contributors right now, but someday I'd love to upgrade that to the point where someone can use it to learn Roc and or experiment with different platforms.

It always a bit of a challenge helping new contributors find meaningful work, when everyone has such different backgrounds and interest. If you feel like posting in introductions and tell us a bit about yourself, that could be helpful and is a great way to find others in the Roc community with similar interests.

I'm usually experimenting with platform things, and love to hack things together to explore new ideas. I've been incrementally levelling up my compiler knowledge by picking up things that need doing. I've had some success just looking at the "TODO" comments in the compiler before, and then asking people about the next steps.
