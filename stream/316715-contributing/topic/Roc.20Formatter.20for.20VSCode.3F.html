Shritesh Bhattarai (Oct 10 2022 at 03:23):

I use https://marketplace.visualstudio.com/items?itemName=emeraldwalk.RunOnSave with the following json settings to format on save:

    "emeraldwalk.runonsave": {
        "commands": [
            {
                "match": "\\.roc$",
                "cmd": "roc format ${file}"
            }
        ]
    }
