Topic: alpha3

Anton (Feb 26 2025 at 13:17):

Roc alpha3 is out. It mainly solves some macos issues. There are no new features because we're working on a clean rewrite of the compiler in zig that aims to be reliable and easy to maintain :)

Breaking Changes

  • Both macos releases are now built with macos 13 so they may no longer work on macos 12.
  • old_linux releases are gone, they were built with really outdated software at this point.
  • 0- was removed from the filename of the tar.gz archives.
  • 0.0.0- was removed from the tag name; it is now alpha3-rolling

Other Changes

  • The roc binaries built for macos no longer require the z3 dependency. brew install zstd should be all that is required.
